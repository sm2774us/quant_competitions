    📋 Electronic Trading Challenge

    Participants must develop a market-making bot for a simulated electronic exchange.

    Problem Description

    • Market Making: Providing liquidity by posting both buy and sell orders.
    • Fair Value Estimation: Calculating the true value of an asset based on order book signals.
    • Risk Management: Managing position limits and hedging market risk.

    Solutions

    • Python 3.13
    • C++26
    • Rust 1.93.1

    📈 Market Prediction

    A data science challenge focused on predicting future market movements from a set of anonymous features.

    Problem Description

    • Time-Series Analysis: Handling temporal dependencies in market data.
    • Inference Latency: Models must be efficient enough for real-time application.
    • Evaluation: Scored on Utility score (weighted sum of returns).

    Solutions

    • Python 3.13
    • C++26
    • Rust 1.93.1
